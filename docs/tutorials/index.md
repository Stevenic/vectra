# Tutorials
{: .no_toc }

End-to-end walkthroughs for common Vectra use cases.
{: .fs-6 .fw-300 }

---

These tutorials take you from zero to a working application. Each one builds on the concepts covered in the main documentation — if you haven't read the [Getting Started](/vectra/getting-started) guide yet, start there.

| Tutorial | What you'll build |
|----------|-------------------|
| [Build a RAG Pipeline](/vectra/tutorials/rag-pipeline) | Ingest documents, query by text, render context sections, and feed them to an LLM |
| [Custom Storage Provider](/vectra/tutorials/custom-storage) | Implement the `FileStorage` interface with SQLite as the backing store |
| [Browser & Electron App](/vectra/tutorials/browser-electron) | Semantic search in the browser with IndexedDB, TransformersEmbeddings, and no API key |
| [Cross-Language with gRPC](/vectra/tutorials/cross-language) | Start the gRPC server and query Vectra from Python |
| [Auto-Sync a Folder](/vectra/tutorials/folder-sync) | Keep an index in sync with a directory of files using FolderWatcher and the CLI |
